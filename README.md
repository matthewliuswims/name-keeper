### To run app locally:
```
# Make sure RNDebugger is open and wait state. then run the below command to start app

REACT_DEBUGGER="unset ELECTRON_RUN_AS_NODE && open -g 'rndebugger://set-debugger-loc?port=19001' ||" npm start
```
